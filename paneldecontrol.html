Registro de Productos

El módulo permite agregar, editar y eliminar productos en el sistema web. Cuenta con un formulario para ingresar datos como Codigo, nombre, descripción y precio.

Inventario

Permite llevar el control de stock de los productos mediante ajustes por entradas y salidas. Cuenta con alertas de productos por agotarse y genera reportes del inventario.

Control y Resporte Ventas

Permite generar y gestionar pedidos, facturas y órdenes de venta. Se integra con el inventario para descontar productos vendidos.

Ventas y facturacion

Permite generar órdenes y facturas de venta, calculando impuestos y totales automáticamente.
